# MBTI Pixel Art Characters

วางไฟล์ pixel art ตัวละคร MBTI ไว้ในโฟลเดอร์นี้

## ชื่อไฟล์ที่แนะนำ

ใช้ชื่อ MBTI type เป็นตัวพิมพ์ใหญ่ และใช้ `.png` พื้นหลังโปร่งใส

```text
INTJ.png
INTP.png
ENTJ.png
ENTP.png
INFJ.png
INFP.png
ENFJ.png
ENFP.png
ISTJ.png
ISFJ.png
ESTJ.png
ESFJ.png
ISTP.png
ISFP.png
ESTP.png
ESFP.png
```

## ขนาดที่แนะนำ

- ใช้ไฟล์ `.png`
- พื้นหลัง transparent
- ขนาดต้นฉบับ `512x512` หรือ `1024x1024` ได้
- ถ้าทำเป็น pixel asset จริง แนะนำ export อีกชุดเป็น `128x128`

ตอนนี้เว็บยังไม่ดึงรูปขึ้นหน้า result อัตโนมัติ เพื่อป้องกันภาพหายแล้วเกิด broken image หลังจากใส่รูปครบหรือใส่บางส่วนแล้วค่อยเพิ่ม logic แสดงรูปแบบมี fallback ได้
